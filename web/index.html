Features

Built to be used, not demoed.

Every feature here was written because someone needed it. Nothing was added for a press release.

Network-level ad blocking
Blocks at the WebView2 request layer before anything leaves the machine. StevenBlack hosts list, refreshed every 7 days. On by default.
No accounts, no telemetry
Nothing phones home. No analytics, no crash reports. Your browsing history is a file on your own machine.
Chrome extensions
Load any unpacked Chrome extension from a local folder. Manifest V2 and V3 both work. Each extension gets an Open button for its popup.
Mica glass window
DWM Mica and Acrylic backdrop. The Pegboard canvas inherits the window backdrop so tiles visually float on the desktop.
Low-level keyboard hook
Shortcuts intercept before Chromium. They fire from inside pages, text fields, and captchas. No exceptions.
Download manager
Floating dropdown panel with live progress bars, open and dismiss buttons, and a badge on the toolbar showing in-progress count.
Privacy

Your data is a file on your own machine.

Bookmarks, history, and settings live at %LocalAppData%\Flint\profile.json. Nothing leaves without you navigating somewhere.

  • No analytics or crash reporting
  • No account sign-in or cloud sync
  • No background update processes
  • No AI sidebar or suggestions
What does connect

Only when you use it.

These are the only outbound connections Flint ever makes.

  • Pages you navigate to
  • Favicon lookups via Google (for tab icons)
  • Ad-block list refresh from GitHub, every 7 days
  • Open-Meteo weather, only if you add a weather tile
