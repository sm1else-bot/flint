The best extensions for Flint.

Flint loads any unpacked Chrome extension from a local folder. These are the ones worth having. Each one is open source and links directly to its GitHub releases page.

How to install an extension in Flint
Download the extension release zip, extract it to a folder, then add it in Flint's settings.
  1. Download the release zip from the extension's GitHub releases page
  2. Extract the zip to a permanent folder on your machine
  3. Open Flint Settings (Ctrl+,) and go to Features > Extensions
  4. Click "Add from folder" and select the extracted folder
  5. The extension loads immediately and persists across restarts
