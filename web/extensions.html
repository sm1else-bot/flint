Extensions

The best ones, with direct links.

Flint loads any unpacked Chrome extension from a local folder. These are the ones worth having. All open source, all linking directly to GitHub releases.

How to install an extension in Flint
  1. Download the release zip from the extension's GitHub releases page below
  2. Extract it to a permanent folder on your machine
  3. Open Flint Settings with Ctrl, and go to Features, then Extensions
  4. Click "Add from folder" and select the extracted folder
  5. Extension loads immediately and persists across restarts
